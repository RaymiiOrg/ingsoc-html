OpenSSL Decoder Script

Published: 15-11-2014 | Author: Remy van Elst | Text only version of this article


Table of Contents

Simple PHP script which decodes an SSL connection or certificate and displays information.

Features

  • Decodes CSR
  • Decodes Certificates
  • Decodes SSL Connections
  • OCSP validation
  • CRL validation
  • Full certificate chain validation.
  • Issuer validation
  • Date validation

Requirements

  • PHP 5.6+
  • OpenSSL
  • PHP must allow shell_exec and remote fopen.
